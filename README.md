# Personal Resume Website

This project is a personal resume website created as part of the Web Development course assignment.

The website presents my personal information, education background, skills, projects, and contact details in a simple and responsive design.

## Features

- Personal information and career objective
- Education details
- Skills section
- Projects section (minimum of two projects)
- Contact information
- Responsive design for different screen sizes
- Bootstrap Navbar component

## Technologies Used

- HTML5
- CSS3
- Bootstrap

## Project Purpose

The purpose of this project is to demonstrate my understanding of web development fundamentals, including creating structured web pages, applying styling using CSS, and designing responsive layouts.

## Bonus Implementation

- The website was uploaded to GitHub Pages.
- Bootstrap Navbar component was used to improve navigation and responsiveness.
